Web Development-ის კომპონენტები

  • Frontend ( ვიზუალური ნაწილი, რასაც მომხმარებელი ხედავს )
  • Backend ( სერვერის მხარე, მონაცემთა დამუშავება )
  • Database ( მონაცემთა ბაზები, ინფორმაციის შენახვა )
  • Hosting / Server ( ვებსაიტის განთავსება ინტერნეტში )

Frontend-ში გამოყენებული ენები

  • HTML ( საიტის სტრუქტურა )
  • CSS ( სტილები, დიზაინი )
  • JavaScript ( ინტერაქტივობა )
